A) KEUANGAN SUITE (Laba/Rugi/Modal)
Source: app/Http/Controllers/ReportController.php

1) Revenue
Formula:
Revenue = sum(total_harga) for jenis_transaksi = penjualan

2) HPP (Harga Pokok Penjualan)
Formula:
HPP = sum(cost_per_unit * jumlah)
where cost_per_unit = batch.harga_beli OR obat.harga_beli OR 0

3) Laba Kotor
Formula:
GrossProfit = Revenue - HPP

4) Biaya Operasional
Formula:
OperatingExpense = RentExpense + OtherOperatingExpense
(Only kategori sewa + lainnya)

5) Laba Operasional
Formula:
OperatingProfit = GrossProfit - OperatingExpense

6) Laba Bersih
Formula:
NetProfit = OperatingProfit - TaxExpenseApplied - InterestExpenseApplied

7) Modal Barang Aktif
Formula:
ModalBarang = sum(stok_tersedia * harga_beli) for stok_tersedia > 0

8) Expense toggle logic
TaxExpenseApplied = include_pajak ? taxExpense : 0
InterestExpenseApplied = include_bunga ? interestExpense : 0

B) MARGIN (Operasional Cerdas)
Source: app/Actions/BuildOperationalInsights.php

Per item and per batch:

1) Revenue
Formula:
Revenue = sum(total_harga)

2) Cost
Formula:
Cost = sum(buy_price * jumlah)
where buy_price = batch.harga_beli OR obat.harga_beli OR 0

3) Margin value
Formula:
MarginValue = Revenue - Cost

4) Margin percentage
Formula:
MarginPct = (MarginValue / Revenue) * 100, if Revenue > 0 else 0

5) Summary 30 days
- revenue_30d = sum(perItem.revenue)
- cost_30d = sum(perItem.cost)
- margin_value_30d = sum(perItem.margin_value)
- avg_margin_percentage_30d = average(perItem.margin_percentage)

C) CASHFLOW SUITE
Source: app/Http/Controllers/ReportController.php

1) Cash in components
- CashInFromMasuk = sum(total_harga where jenis_transaksi = masuk)
- CashInFromDirectSales = sum(total_harga where jenis_transaksi = penjualan AND hutang is null)
- CashInFromHutangPayments = sum(hutang_payments.amount in selected date range)

2) Total Cash In
Formula:
CashIn = CashInFromMasuk + CashInFromDirectSales + CashInFromHutangPayments

3) Cash Out
Formula:
CashOut = sum(total_harga where jenis_transaksi = keluar)

4) Net Cashflow
Formula:
NetCashflow = CashIn - CashOut

5) PPN collected from sales
Formula:
PPNCollected = sum(total_harga * ppn_percent/100) for taxed sales
Default ppn_percent = 11
Override source: parsed from keterangan text pattern "PPN: x%"
